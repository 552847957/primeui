Name Type Default Description
caption string null Name for the list.
choices string , number null Choices for the list.
items string , number null Pointing the subgroups of choices.
effect boolean false Effect of subgroups to appear.
showheader boolean false Headers of subgroups.

No Events.

Name Parameters Description
disable - Disables the list container.

Example

            $('#single').puimultiselectlistbox('disable');
