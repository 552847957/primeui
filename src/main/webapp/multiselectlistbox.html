            $('#single').puiselectbutton({
                choices: [
                   {label:'Xbox One', value:0},
                   {label:'PS4', value:1},

                ]

Name Type Default Description
choices string , number null Choices for the buttons.
formfield string null Name for the form.
unselectable boolean false State for the unselectable buttons.
tabindex string '0' Order of elements in tabbing.
multiple boolean false State for multiple button selection.
Name Parameters Description
change event: puiselectbuttonchange event
Fires when value changes.

Example

            $('div').puiselectbutton({
                change: function(event) {
                    //..
            }});
Name Parameters Description
selectOption value Selects the clicked option.
unselectOption value Unelects the clicked option.

Example

            $('#single').puiselectbutton('selectOption',0);
                                
$(function() {
    $('#single').selectbutton({
                formfield: 'single_input',
                choices: [
                   {label:'Xbox One', value:0},
                   {label:'PS4', value:1},
                   {label:'Wii', value:2},
                   {label:'PS3',value:3}
                ]
            });
            
            $('#multiple').selectbutton({
                formfield: 'multiple_input',
                choices: [
                   {label:'Xbox One', value:0},
                   {label:'PS4', value:1},
                   {label:'Wii', value:2},
                   {label:'PS3',value:3}
                ],
                multiple: true,
                unselectable: true
            });
});
