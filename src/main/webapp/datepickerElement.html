Usage

Datepicker element is defined usign p-datepicker tag. 
<p-datepicker></p-datepicker>

Attributes

Name Type Default Description
weekHeader string "Wk" The text to display for the week of the year column heading.
dateFormat string "mm/dd/yy" The format for parsed and displayed dates.
firstDay number 0 Set the first day of the week.
rtl boolean false Whether the current language is drawn from right to left.
showMonthAfterYear boolean false Whether to show the month after the year in the header.
yearSuffix string "" Additional text to display after the year in the month headers.
showOn string "focus" When the datepicker should appear.
showAnim string "show" The name of the animation used to show and hide the datepicker.
defaultDate data,number,string null Set the date to highlight on first opening if the field is blank.
appendText string "" The text to display after each date field.
buttonText string "..." The text to display on the trigger button.
buttonImage string "" A URL of an image to use to display the datepicker when the showOn option is set to "button" or "both".
buttonImageOnly boolean false Whether the button image should be rendered by itself instead of inside a button element.
hideIfNoPrevNext boolean false Normally the previous and next links are disabled when not applicable. You can hide them altogether by setting this attribute to true.
navigationAsDateFormat boolean false Whether the currentText, prevText and nextText options should be parsed as dates by the formatDate function, allowing them to display the target month names for example.
gotoCurrent booelan false When true, the current day link moves to the currently selected date instead of today.
changeMonth boolean false Whether the month should be rendered as a dropdown instead of text.
changeYear boolean false Whether the year should be rendered as a dropdown instead of text.
yearRange string "c-10:c+10" The range of years displayed in the year drop-down.
showOtherMonths boolean false Whether to display dates in other months at the start or end of the current month.
selectOtherMonths boolean false Whether days in other months shown before or after the current month are selectable.
showWeek boolean false When true, a column is added to show the week of the year.
calculateWeek function() jQuery.datepicker.iso8601Week A function to calculate the week of the year for a given date.
shortYearCutoff number,string "+10" The cutoff year for determining the century for a date.
minDate number,date,string null The minimum selectable date.
maxDate number,date,string null The maximum selectable date.
duration string "normal" Control the speed at which the datepicker appears.
beforeShowDay function(Date date) null A function that takes a date as a parameter and must return an array.
beforeShow function( Element input, Object inst ) null A function that takes an input field and current datepicker instance and returns an options object to update the datepicker.
onSelect function( String dateText, Object inst ) null Called when the datepicker is selected.
onChangeMonthYear function( Integer year, Integer month, Object inst ) null Called when the datepicker moves to a new month and/or year.
onClose function( String dateText, Object inst ) false Called when the datepicker is closed.
numberOfMonths number,array 1 The number of months to show at once.
showCurrentAtPos number 0 When displaying multiple months via the numberOfMonths option, the showCurrentAtPos option defines which position to display the current month in.
stepMonths number 1 Set how many months to move when clicking the previous/next links.
stepBigMonths number null -
altField selector,jquery,element "" An input element that is to be updated with the selected date from the datepicker.
altFormat string "" The dateFormat to be used for the altField option.
constrainInput boolean true When true, entry in the input field is constrained to those characters allowed by the current dateFormat option.
showButtonPanel boolean false Whether to display a button pane underneath the calendar.
autoSize boolean false Set to true to automatically resize the input field to accommodate dates in the current dateFormat.
disabled boolean false -

Methods

Name Parameters Description
enable() - Enables the element.
disable() - Disables the element.
destroy() - Destroys the element.
