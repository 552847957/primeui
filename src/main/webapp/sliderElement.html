Usage

Slider is an extension to standard jquery ui slider widget using <p-slider></p-slider>. 
<p-slider></p-slider>

Attributes

Name Type Default Description
animate boolean false Animation of the element.
distance number 0 Distance!!
min number 0 Mininum value for slider.
max number 100 Maximum value for slider.
orientation string horizontal Determines whether the slider handles move horizontally or vertically
range boolean false Whether the slider represents a range.
step number 1 Determines the size or amount of each interval or step the slider takes between the min and max.
value number 0 Determines the value of the slider.
values array null This option can be used to specify multiple handles.

Methods

Name Parameters Description
enable() - Enables the element.
disable() - Disables the element.
destroy() - Destroy the selected element.
