Attributes

Name Type Default Description
autoDisplay boolean true Displays submenus without the mouse click
target selector null Target element to attach the menu, can be a jQuery selector or document object

Functions

Name Parameters Description
show - Shows Menu.
hide - Hides Menu.
getTarget - Gets the menu target.
isVisible - Returns visibility state of menu.

Javascript

$('#cm').puicontextmenu({
target: $('#img1')
});
