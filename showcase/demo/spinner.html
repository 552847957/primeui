Spinner Spinner is an input component to provide a numerical input via increment and decrement buttons.

<input id="default" type="text" />

Note: Spinner requires jquery mousewheel plugin for mousewheel support.

Name Type Default Description
step number 1.0 Step factor to increment/decrement.
min number null Mininum value for spinner.
max number null Maximum value for spinner.
prefix string null Prefix for the value of spinner.
suffix string null Suffix for the value of spinner.

No events.

Name Parameters Description
enable - Enable the widget
disable - Disable the widget.
option name: Name of the option Returns the value of the option.
option name: Name of the option, value: Value of the option Set the value of the option. 
$(function() {
    $('#basic').puispinner();

    $('#step').puispinner({step:0.25});

    $('#minmax').puispinner({
        min:0,
        max:100
    });

    $('#prefix').puispinner({prefix:'$'});                                           
});
