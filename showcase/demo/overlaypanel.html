OverlayPanel OverlayPanel is a generic container component that can overlay other components on page. Notable features are custom positioning, configurable events and effects. Lazy content loading to reduce page load time is also supported via dynamic option, when enabled overlayPanel will load the contents just before being shown.

Name Type Default Description
target string null Identifier of the target component to show the panel.
showEvent string click Event to show the overlay panel.
hideEvent string click Event to hide the overlay panel.
showCloseIcon boolean false Displays a close icon to hide the overlay.
dismissable boolean false When set true, clicking outside of the panel hides the overlay.
my string left top Position of the panel relative to the target.
at string left bottom Position of the target relative to the panel.
Name Parameters Description
onShow - Client side callback to execute when panel is displayed.
onHide - Client side callback to execute when panel is hidden.

Example

            $('#default').puioverlaypanel('onShow');
Name Parameters Description
disable - Disables the button.
enable - Enables the button.
show - Shows the container.
hide - Hides the container.

Example

            $('#default').puibutton('disable');
